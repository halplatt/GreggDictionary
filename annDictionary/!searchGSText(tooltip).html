G-S-T Quick Reference

Simply type Gregg stroke names separated by dashes (e.g., t-e-n) and press Enter. Ambiguities will be auto-corrected (e.g., k/c becomes k, df/dv/tf/tv becomes df).

Standard Names

k, g, r, l, n, m, t, d, p, b, f, v, j, s, x

Strokes

th, ch, sh, ng, nk, ld, rd, ss, xs

Blends

nt→nt,nd · mt→mt,md · tn→tn,dn · tm→tm,dm · mn→mm,nm
td→td,dt,dd · df→df,dv,tv · jnt→jnt,jnd,pnt,pnd

Special Vowel Forms

  • ie: circle inside i → as in h-ie highly
  • ea: dot inside a → as in p-ea-n-o piano
  • Hook vowels: o, u
  • Circle vowels: a, e, i

Special Characters

  • h → represents a, h, or ing dot (e.g., h-e-r-h herring)
  • / → disjoined (e.g., a-tn/k identical)
  • -/- →intersected (e.g., k-/-t quantity)
  • w → underline vowels: (e.g., k-w-e-n queen )
  • ya, ye → make circle a loop (e.g., e-s-ye easily)

Indicators (if specified 1=clockwise 2=counterclockwise)

Digit rule: Digits are ignored (a matches a, a1, a2) unless you include a digit in your search.

  • reverse circle (a2, e2, i2)
  • th2 → under-th
  • S1 → left-s
  • S2 → right-s
